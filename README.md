# Rush Horizon

Rush Horizon is a 3D endless runner mobile game developed in Unity.

## Features

- Endless Runner Gameplay
- Lane Switching System
- Jump & Slide Mechanics
- Coin Collection
- Power-Ups
- Dynamic Obstacles
- Score System
- Mobile Optimization
